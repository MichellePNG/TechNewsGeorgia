📌 About TechNews Georgia

TechNews Georgia is a modern web platform designed to deliver the latest news in technology, artificial intelligence, gaming, cybersecurity, and innovation.

The goal of this project is to demonstrate a full-stack web application using Flask, SQLAlchemy, and dynamic HTML templates. Users can create, read, update, and delete news articles easily.

🚀 Features

  • 📰 Latest Technology News
  • ➕ Add new articles
  • ✏️ Edit existing news
  • 🗑 Delete news
  • 📱 Responsive design

🎯 Purpose

This project was created as a final school assignment to demonstrate backend + frontend integration and database management skills.
